# ia-cloud-nfcpy-id

Read the card ID (Mifare: UID, FeliCa: IDm) with `node-nfcpy-id`.

## Install

Requires the nfcpy module in advance.  
Install the nfcpy module with pip command.  

```
pip install -U nfcpy
```
